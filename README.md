========================================================================
                             MpFourPlayer
========================================================================

Selamat datang di MediaPlayer4U! 

Ini adalah proyek open-source pertama saya. Aplikasi media player ini 
dibuat dengan bantuan Codex untuk mengisi akun GitHub saya agar tidak 
kosong, sekaligus sebagai sarana belajar pemrograman.

------------------------------------------------------------------------
FITUR UTAMA
------------------------------------------------------------------------
* Memutar berkas audio dan video.
* Antarmuka yang simpel dan mudah digunakan.
* Kode sumber terbuka (open-source) untuk dipelajari bersama.

------------------------------------------------------------------------
TEKNOLOGI YANG DIGUNAKAN
------------------------------------------------------------------------
* Kode dasar dibuat menggunakan Codex.
* Bahasa Pemograman: C++.

------------------------------------------------------------------------
CARA MENJALANKAN
------------------------------------------------------------------------
1. Unduh atau klon (clone) repositori ini.
2. 2. Buka berkas solusi bernama **`.slnx`** menggunakan:
   * **VS Code** (pastikan ekstensi C/C++ yang sesuai sudah terpasang).
   * **Visual Studio** versi terbaru.
   * Aplikasi compiler lain yang kompatibel dengan format `.slnx`.
3. Jalankan atau *compile* proyek langsung dari aplikasi tersebut.
